Idea:
 App that gives information on all the live fixtures on any tv channel

 Know all the live fixtures on any sporting events.
Put a reminder of game and result notifications
Applicable for all cable channel users
 
Targeted audience:
 All sports lover. Any sports lover
 
Main Benefits:
User can know which game they can watch at any time. User can put a reminder to remind them of the game
