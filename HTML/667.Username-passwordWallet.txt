Username-passwordWallet

Submitted on:  September 13, 2015 - 9:57am 
 
Idea:
 For removing confusion of user, for a given site which username and password did he used to login.
 
Problem Solved:
Their are so many social networking sites we using in our daily life. For easy we keep same username and password for all the site. If one finds your username and password of one site he can easily get excess over other sites. And people keep forgeting what their username and password was for a given site so they  keep creating account again and again which is totally irritating and time consuming. So our app deals about this problem of every user.
 
 
Functions:
 Our app will list for a given site which user ID and password did you used to login in totally secured way.
 
Targeted audience:
 All the user who are using Social Networking and other popular sites.
 
Main Benefits:
User don't have to remember what their username and password was for the given site. They can try new username and password  without hesitating.
 
Platform: Android, Windows phone
